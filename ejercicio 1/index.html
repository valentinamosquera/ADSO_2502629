git

Git is the open source distributed version control system that facilitates GitHub activities on your laptop or desktop. This cheat sheet summarizes commonly used Git command line instructions for quick reference

Comandos principales de git

  • Cd:Verificamos la carpeta que estamos, Ingresa a un directorio(Carpeta) a una terminal
  • Git INIT:Iniciar un proyecto git.
  • Git config --global user.email "Email personal":Identificar y configurar localmente el email en un repositorio local
  • Git config --global user.name "name personal":Identificar y configurar localmente el nombre del usuario en un repositorio local
  • Git add . :Agrega todos los archivos modificados al reporsitorio
  • Git status:Muestra el estado del directorio de trabajo y permite ver los cambios que se han preparado
  • Pwd Muestra la ruta en donde estoy parado, Mostrar la ruta de directorio actual en la terminal
  • Cd "Nombre archivo"Sirve para ingresar a una carpeta o documento
  • clearBorrar pantalla del terminado
  • Git config user.nameIdentifica el nombre de ususario en el repositorio local
  • Git config user.emailVerifica el correo de usuario en el reporsitorio local
  • Git config --global -- user-all user.name:Elimina las configuraciones del nombre de usuario en el repositorio
  • lSIngresar a una ruta especifica(Muestra la lista de contenidos de un directoriol)
